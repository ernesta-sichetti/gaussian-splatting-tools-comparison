# Contributing to Gaussian Splatting Tools Database

Thank you for helping improve this repository!

## How to add a tool

1. Edit `trainers.md` or `viewers.md`
2. Do **not remove any columns**. Keep the first column as **Tool Type**.
3. Add **Tags** in the same comma-separated format.
4. Fill the **Validation** column:
   - Tested by author
   - Declared (docs)
   - Community-reported
   - Not verified
5. Include Notes if needed (extra info, links, limitations).

## Submitting changes

1. Fork this repository  
2. Create a new branch for your edits  
3. Make your changes  
4. Open a Pull Request describing the updates  

Thank you for contributing!
